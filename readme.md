# Node-Weather

Testing node request with OpenWeather.org API.

## Setup
```
$ git clone https://github.com/royboy5/node-weather.git <folder name>
$ cd <folder name>
$ npm i
```

## Usage

Get weather using current location

```
$ node app.js
```

Get weather by specifying city

```
$ node app.js <city>

Example

$ node app.js -l 'New York'

```
